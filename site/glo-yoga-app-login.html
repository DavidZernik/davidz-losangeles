Glo

Yogaglo website and application

From 2016 - 2108, I was a front-end developer at Glo, and developed the UI for YogaGlo's web application, a full-stack platform built with Ember.js and Express.

Back in 2016, streaming exercise classes online was cutting edge!

I worked on multiple features including the login page highlighting their "Notes" feature, user dashboard, and class browsing interface. Built responsive, mobile-first interfaces using HTML, CSS, and JavaScript that performed consistently across all devices and browsers. Focused on creating an intuitive user experience for yoga practitioners to access classes, save personal notes, and filter content by their preferences.
