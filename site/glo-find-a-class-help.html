Glo

Email to help users find a class

This project focused on building a personalized “find a class” email for YogaGlo, designed to help users quickly discover content that matched their interests and experience level. I worked with a machine learning engineer who owned the recommendation logic, and my role was to integrate those results into a dynamic email template that could render different class cards per user. The front end was built with responsive, email-safe HTML and CSS to keep the layout clean and scannable across devices.

On the implementation side, the template pulled recommendations from an internal endpoint and mapped them into reusable content blocks. I tested the build across major email clients to make sure images, spacing, and stacking behavior held up. The result was a scalable, personalized email that turned raw recommendation data into a usable product experience inside the inbox.
