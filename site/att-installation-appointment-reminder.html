AT&T

Service message email

This project was a service message email for AT&T, where I handled the build and CRM integration using HTML, CSS, and Salesforce Marketing Cloud with AMPscript for personalization. I worked with marketers and data partners to make sure variable content like names, dates, and time windows rendered correctly and stayed readable across devices, with a strong focus on accessibility and real-world inbox behavior.

Technically, I built a responsive layout with image swaps for mobile, added alt text for ADA compliance, and tuned image weight and text balance to protect deliverability. I tested across Outlook, Gmail, and mobile clients, wired in dynamic fields from the sendable data, and delivered a stable, reusable template that could support high-volume, time-sensitive sends without breaking.
