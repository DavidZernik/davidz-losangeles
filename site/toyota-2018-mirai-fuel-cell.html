Toyota

Toyota Mirai Email Campaign

Developed responsive HTML email template for Toyota's Mirai hydrogen fuel cell vehicle campaign promoting zero-emission mobility technology. Collaborated iteratively with design team to address critical accessibility and cross-client compatibility issues including converting text-embedded images to live HTML for screen reader support, replacing gradient backgrounds causing Outlook rendering failures with solid colors, and implementing mobile-responsive stacking for side-by-side layouts.

Built bulletproof CSS buttons replacing image-based CTAs for universal client compatibility. Added comprehensive alt-text attributes for all imagery to ensure ADA compliance. Implemented feature callouts showcasing $15,000 fuel credit, CA HOV carpool sticker eligibility, and extended driving range with icon-based design elements and social media integration.
