Glo

Email up-selling features

This project focused on building a feature education and upsell email for YogaGlo that highlights cross-device use, offline class downloads, and Apple Watch support. I built the front end email in responsive HTML and CSS and implemented it within Iterable, translating the product and design concepts into a layout that holds up across major email clients while clearly communicating the value of each feature.

Technically, the work centered on email-safe layout patterns and defensive coding for inconsistent client rendering. I tested and refined the template in clients like Gmail and Outlook to keep spacing, typography, and stacking behavior consistent on mobile and desktop. The result was a lightweight, reusable template the marketing team could rely on for future feature announcements.
