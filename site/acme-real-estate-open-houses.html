ACME

Marketing email for real estate firm

This project was a marketing email for ACME Real Estate, built to showcase current listings and promote upcoming open houses in a clean, image-forward layout. The marketing team had an existing template direction but needed help turning it into something flexible and reliable for real campaigns. I stepped in to debug the build, finish the front end development, and adapt the template so it could handle changing property data without breaking the layout or visual hierarchy.

From an implementation standpoint, the work focused on responsive, email-safe HTML and CSS and careful handling of image-heavy sections. I tested the template across major email clients to make sure spacing, typography, and grid behavior held up on both desktop and mobile. The end result was a polished, reusable template the team could use for ongoing property and open house announcements.
