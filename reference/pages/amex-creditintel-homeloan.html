American Express Financial Calculators - Home Loan

​​

I was the lead developer across a suite of about seven financial calculators for American Express Credit Intel, including the Home Equity Calculator. Each tool was built with HTML, CSS, and JavaScript, with a shared focus on clear UX, responsive layout, and accessibility so the experiences worked consistently across devices and input methods.

​

For the Home Equity Calculator specifically, I implemented the UI and calculation logic that estimates available equity from home value and mortgage balance, with strong input validation and edge case handling for scenarios like low or negative equity. More broadly, I helped design a reusable pattern for these tools so calculations, formatting, and error handling stayed consistent across the entire calculator suite as requirements and compliance reviews evolved.
